One Command Install

curl | bash. Installs psutil, deploys agent, creates systemd service. Done in 30 seconds.

Offline Buffer

Connection lost? Agent buffers up to 100 readings and sends them when reconnected.

Open REST API

Query your server metrics programmatically

# List all monitored servers
GET /api/v1/servers
# Get metrics for last 24h
GET /api/v1/servers/{id}/metrics?hours=24
# Get current health status
GET /api/v1/servers/{id}/status
# Try it right now:
curl /api/v1/servers | jq
